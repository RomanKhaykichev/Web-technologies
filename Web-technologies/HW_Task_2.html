HTML - CSS

HTML - HyperText Markup Language «язык гипертекстовой разметки»
- для просмотра веб-страниц в браузере.
Веб-браузеры получают HTML документ от сервера по протоколам HTTP/HTTPS

CSS - Cascading Style Sheets «каскадные таблицы стилей»
- формальный язык декорирования и описания внешнего вида документа (веб-страницы),
написанного с использованием языка разметки (чаще всего HTML или XHTML).

HTML - Теги

< > - openning tag
< / > - closing tag

Теги HTML упорядочены по категориям:
  • Базовый HTML-код
  • Форматирование
  • Формы и ввод
  • Рамки
  • Изображения
  • Аудио/видео
  • Ссылки
  • Списки
  • Таблицы
  • Стили и семантика
  • Мета Инфо
  • Программирование
Полную информацию можно посмотреть на сайте - HTML5CSS.ru
